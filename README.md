# Greenity
_Awesome WM config file based on the Holo Theme (v4.2)_

## Mandatory deps
- [x] luasocket

## Dependencies required 
- [x] mpc
- [x] mpd
- [x] xset         
- [x] urxvt        
- [x] scrot        
- [x] dmenu        
- [x] Roboto       
- [x] i3lock
- [x] ncmpcpp
- [x] unclutter    
- [x] consolekit
- [x] FontAwesome  

## Features
_Others_
- Binary Tree layout ([Treetile](https://github.com/guotsuan/awesome-treetile)

_Keybindings_
- Block screen: F12
- Printscreen¹ PrtSc 
- Increase volume: F3 
- Decrease volume: F2 
- Treetile vertical split set: Super + v
- Treetile horizontal split set: Super + h
